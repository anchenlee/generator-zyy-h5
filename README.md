# 基于Webpack和Gulp的H5小项目生成器
> H5 日益普及，为了方便日后开发的方便，特此写一个模板生成器为日后使用

## 使用yoeman创建模板
安装 yo
```
npm install -g yo
```
新建一个项目目录并进入
```
mkdir my-new-h5 && cd $_
```
安装H5模板生成器
```
npm install -g generator-zyy-h5 
```
使用H5模板
```
yo zyy-h5
```

## 使用yoeman创建模板
开发
```
gulp watch
```

发布
```
gulp release
```
